Welcome to this Flask App

This App is used for educational purposes, and showcases:

  1. A Flask app structure split into app and views
  2. How to use Flask to show Python content
  3. How to work with user inputs
  4. How to use Flask to help filter results
There is information written on pages, and on comments within the code. Information on pages regarding the code will be written as such: 
    A Flask app is built from the Flask class. It can be structured as:

         - Main Folder
            - static (a folder for things like images and css files)
                - style.css 
            - templates (a folder for things like HTML files)
                - base.html 
                - index.html 
                - page2.html 
            - app.py (The main file for creating the app)
            - views.py (The file that defines which url triggers which function)
The app also contains a requirements.txt file. This is used for deployment services like Render, and outlines what needs to be installed for the app to work. These are called dependencies because the app is dependent on them. The app also contains a .gitignore file which lists files that are on my local version (my computer) that I don't want commited to Git or GitHub.
